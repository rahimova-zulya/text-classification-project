# text-classification-project
Multi-class text classification of Russian news (Culture, Sports, Economics). Benchmarking 7 ML/DL models: LogisticRegression, SVM, RandomForest, NaiveBayes, KNN, FastText, BERT. Extensive EDA, hyperparameter tuning, and visualization. Best model: LogReg + TF-IDF achieves F1-macro 0.9875.
